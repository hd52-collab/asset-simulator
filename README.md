# 資産シミュレーター

55歳からの資産推移をシミュレーションするWebアプリです。

## 機能

- **初期資産**：投資信託 2,700万・高配当株 1,500万・現金（選択可）
- **運用前提**：高配当株配当利回り 3.5%固定、公的年金 65歳から月15万円
- **調整パラメーター**：投資信託年利・月の生活費・年間特別支出・取り崩し設定
- **グラフ**：積み上げ棒グラフ（投資信託・高配当株・現金）＋総資産折れ線＋目標2,000万点線

## 使い方

1. ページ上部のボタンで現金初期値・投資信託年利を選択
2. スライダーで月の生活費・年間特別支出を調整
3. 取り崩し設定（金額・開始・終了年齢）を選択
4. 65 / 75 / 85 / 95歳時点の総資産カードとグラフがリアルタイムで更新されます
