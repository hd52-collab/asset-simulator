## 資産シミュレーター（asset_simulator.html）

55歳からの資産推移をシミュレーションするWebアプリです。

### 機能

- **初期資産**：投資信託 2,700万・高配当株 1,500万・現金（選択可）
- **運用前提**：高配当株配当利回り 3.5%固定、公的年金 65歳から月15万円
- **調整パラメーター**：投資信託年利・月の生活費・年間特別支出・取り崩し設定
- **グラフ**：積み上げ棒グラフ（投資信託・高配当株・現金）＋総資産折れ線＋目標2,000万点線

### 使い方

1. ボタンで現金初期値・投資信託年利を選択
2. スライダーで月の生活費・年間特別支出を調整
3. 取り崩し設定（金額・開始・終了年齢）を選択
4. 65 / 75 / 85 / 95歳時点の総資産カードとグラフがリアルタイムで更新されます

## 中古車コスト比較シミュレーター（car_simulator.html）

2台の中古車を総所有コストで比較するWebアプリです。

### 機能

- **共通設定**：所有年数・年間走行距離・ガソリン単価
- **車A・B個別設定**：車体価格・年式・燃費・年間維持費・車検費用
- **計算内容**：ガソリン代＋車検（2年ごと）＋維持費＋車体価格の総コスト
- **グラフ**：内訳棒グラフ（車A青・車B赤）＋勝者バナー・差額表示

### 使い方

1. 共通設定（所有年数・走行距離・ガソリン単価）をスライダーで調整
2. 車A・車Bそれぞれの価格・燃費・維持費を設定
3. 総コスト・年平均コスト・内訳グラフがリアルタイムで更新されます
